# quotes
Quotes for quote-lovers

# Run
Запускается через server.py на http://127.0.0.1:5010/

# PLAN A:

Приложение должно собирать данные в БД в одну таблицу с 4 полями:
- id цитаты (уникальный)
- дата публикации
- текст цитаты
- рейтинг цитаты
